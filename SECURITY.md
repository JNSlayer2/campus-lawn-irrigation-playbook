# Security Policy

## Supported Scope

This repository contains a static OpenClaw skill and reference policy for
campus lawn irrigation scheduling. It does not contain controller credentials,
runtime logs, device APIs, or site-specific deployment state.

## Do Not Commit

- School names, addresses, or personally identifying information
- Student, family, staff, or visitor data
- Irrigation controller usernames, passwords, tokens, or API keys
- VPN, NAS, tunnel, router, or remote-access details
- On-site photos, screenshots, or exported control logs
- Packaged zip artifacts with unknown extra metadata

## Reporting Security Issues

Open a GitHub issue with a minimal description. Do not include credentials,
personal data, private site details, screenshots, or controller exports.
