# Campus Lawn Irrigation Playbook

這個專案是一個提供給 OpenClaw 使用的草皮噴灌管理 skill，目的是把校園草皮的固定灑水時間與不同情境下的調整方式整理成可重用的規則。

## 這個 skill 在做什麼

此 skill 用來協助判斷：

- 夏季、春季 / 秋季、冬季的草皮噴灌時間
- 不同星期對應的固定灑水時段
- 遇到日常、活動、工程等情境時，應該停灌、延後、縮短時長，或減少開關次數
- 草皮過濕時，如何避免過度灑水並保護草皮狀態

## 內容包含

### 季節排程

- 夏季
- 春季 / 秋季
- 冬季

### 情境調整

- 日常：中午班級曬教具、雨天、溜冰課
- 活動：母親節、校慶、戶外節日活動
- 工程：割草、壓草皮、大型工程車進場、鉅樹工程
- 特殊狀況：草皮過濕，需要調整灑水時間或開關次數

## 回答邏輯

此 skill 會依照以下順序判斷：

1. 先判斷季節
2. 再判斷星期
3. 確認是否有特殊情境
4. 若同時有多個情境，依優先序覆蓋原排程

優先序如下：

`工程 / 安全封鎖 > 雨天或草皮過濕 > 活動或課程占用 > 日常例外 > 季節預設排程`

其他檔案：

- `OpenClaw 啟用指令.txt`：提供安裝後貼給 OpenClaw 的初始化指令
- `安裝說明.txt`：安裝與交付說明

## 安裝方式

1. 找到 OpenClaw 的 `skills` 目錄
2. 將 `campus-lawn-irrigation-playbook` 資料夾複製到該目錄
3. 啟用 skill
4. 將 `OpenClaw 啟用指令.txt` 的內容貼給 OpenClaw

## 公開安全範圍

此 repo 只保留抽象化的草皮噴灌規則、季節排程、情境調整邏輯與
OpenClaw 初始化文字。不要提交校名、地址、學生個資、帳號、內部截圖、
控制器登入資訊、現場照片、匯出的日誌或可直接控制設備的 API/token。

## 適用用途

這個 skill 適合用在：

- 校園草皮灑水規則整理
- 季節性噴灌時段管理
- 活動 / 工程 / 雨天時的灑水調整判斷
- 將固定排程與例外情境整合成標準作業邏輯

## 目前規則假設

- 冬季預設沒有凌晨 `02:00` 噴灌時段
- 若只有部分草皮區域受影響，其他未受影響區域可維持原排程
